This file contains a recipe about making "beef seasoned with soy sauce"(酱牛肉).

酱牛肉

- 牛腱子肉 * 1.5公斤
- 黄豆酱 * 400g
- 酱油(生抽) * 150g
- 酱油(老抽) * 20g
-  焦糖(可以使用白砂糖自制)
- 生姜
-  小葱



香料
- 八角 / star anise
- 桂皮 / cinnamon
- 香叶 / Bay leaf
- 草果 / Black cardamom 
- 小茴香 / cumin
